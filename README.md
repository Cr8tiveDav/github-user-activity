# github-user-activity

A simple Command Line Interface (CLI) to fetch the recent activity of a GitHub user and display it nicely in the terminal.

This project is a task from roadmap.sh [url](https://roadmap.sh/projects/github-user-activity)

## Requirement

[Go](https://go.dev/dl/) **1.21** or higher

## Installation

```bash
# 1. Clone the repository
git clone https://github.com/Cr8tiveDav/github-user-activity.git
```

```bash
# 2. Change directory
cd github-user-activity
```

```bash
# 3. Build binary
go build -o github-user-activity
```

## Usage

```bash
# Check github username activity
./github-user-activity <username>

# help
./github-user-activity -h or --help
```
